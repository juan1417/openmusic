# Openmusic

Openmusic is a Spotify-like prototype built as a Flutter app.

## Scope
- Search and browse music content
- Track playback queue and saved items
- YouTube as catalog/source integration point
- Audio quality selection for owned or licensed content

## Notes
- YouTube content should respect platform rules and available stream qualities.
- FLAC, AAC, and WAV quality options are intended for owned/licensed audio or server-side transcoded assets.
